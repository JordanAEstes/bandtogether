- Node 8.15.x
- MySQL 2.16.x
- MySQL2 1.6.x
- Express 4.16.x
- React 16.8.x
- Sequelize 5.2.x
- Axios 0.18.x
- bcryptjs 2.4.x
- Bootstrap 4.3.x
- Express-session 1.16.x
- Connect-session-sequelize 6.0.0
- Body-parser 1.18.x
- Passport 0.4.x
- UUID 3.3.x
- React-favicon 0.0.14
